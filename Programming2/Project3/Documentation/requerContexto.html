Requerimientos

				

Todo debe de ser guardo en una base de datos:
La Computadora debe de poderse:
	-Crear
	-Consultar
	-Buscar Duennio
	-Borrar
La Computadora debe de tener:
	-Serie
	-Modelo
	-Marca
	-CPU GHZ
	-RAM
	-HDD
	-Sistema Operativo
La Computadora debe de poderse:
	-Consultar
	-Modificar
El Parametro debe de tener:
	-Dias de demora normal
	-Dias de demora urgente
	-Cantidad de solicitudes pendientes máximas de los técnicos
Del Cliente debe de poderse:
	-Crear
	-Buscar por ID
	-Buscar todos
	-Buscar Computadora
El cliente debe tener:
	-ID
	-nombre
	-apellido
	-mail
	-telefono
Del Sistema Operativo debe de poderse:
	-Buscar por ID
	-Buscar todos
	-Crear (funcionalidad extra)
El sistema operativo debe tener:
	-ID
	-Nombre
	-Version
Del software debe poderse:
	-Crear
	-Buscar
	-Buscar por PC
	-Buscar computadoras que no han sido instaladas
El software debe tener:
	-ID
	-Nombre
	-Descripción
	-RAM Requerida
	-Mínimo de HDD
	-Mínimo de CPU
	-Sistema operativo compatible
	-Licencia
	-Fecha Instalación
	-Fecha Expiración	
De la solicitud de instalación se debe poder:
	-Crear
	-Buscar donde Condición es Finalizada/Pendiente
	-Buscar
	-Borrar
